A terminal for running AI coding agents in parallel

Run Claude Code, Codex, and Gemini side by side in a grid and see at a glance which one needs you. macOS, built in Zig on ghostty-vt and SDL3.

The attention problem

Run four agents in four terminal tabs and visually nothing happens: all four tabs look the same. One finishes and sits waiting for your next prompt. Another hits a permission check and waits. You're watching tab 3 and miss both, then twenty minutes later find an agent that's been idle the whole time. Desktop notifications come and go at the OS's whim, and arranging windows by hand doesn't scale. The real problem is that terminals don't know what's running inside them, so a prompt waiting for you looks the same as a command still churning. Architect adds the missing signal.

What it does

Status highlights

A cell changes hue when its agent finishes a task and glows when it's waiting for approval, so you scan the grid and know where to focus. Detection runs on agent hooks, so it works beyond the big three.

Dynamic grid

Start with one terminal in full view. Press ⌘N to add another and the grid expands; close one with ⌘W and it compacts forward. ⌘Enter expands a cell to full screen, ⌘Arrow moves focus, ⌘1–⌘0 jumps to a slot.

Worktree picker (⌘T)

Open a popup to cd into a git worktree for parallel work on separate branches, or create a new one. It works by sending plain git commands to the terminal, fully traceable, no magic.

Recent folders (⌘O)

Jump back into a recently visited directory with instant search filtering. Start typing to narrow the list, with substring highlighting, arrow-key navigation, and ⌘1–⌘9 quick selection.

Diff overlay (⌘D)

Pull up a repo-wide git diff across staged, unstaged, and untracked changes. Click any line to leave an inline comment, then send the whole batch to a running agent (or start one) to act on.

Reader mode (⌘R)

Open a centered markdown reader over the selected terminal's history, with live updates, incremental search, rendered tables and task lists, and clickable links. Useful for long agent output you'd rather read than scroll.

Get it

Early-stage personal project I use daily. macOS only, still with bugs and rough edges.

brew tap forketyfork/architect https://github.com/forketyfork/architect
brew install architect
cp -r $(brew --prefix)/Cellar/architect/*/Architect.app /Applications/
